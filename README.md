# ULS English Website

Website tĩnh dùng để upload lên GitHub Pages.

## Cách dùng
1. Giải nén file ZIP.
2. Upload toàn bộ file và thư mục `assets` lên repository GitHub.
3. Vào Settings → Pages → Deploy from a branch → main → /(root).
4. Chờ 1–3 phút để website chạy.
